Minecraft Server

This is the website for the OMS minecraft server. Most stuff will be found on the discord server though.

Our minecraft server is Fast, Friendly, and Fun!!! We have both a survival world, and a creative world.

Currently our server is running Minecraft (paper) 1.17.1

I currently DO NOT intend to upgrade THIS world to 1.18, however I MIGHT consider making a new world sometime. If I do that, I would have to delete this world.

Plugins List

We run our server on paper and we use these plugins:

  • DiscordChatWM
  • Dynmap
  • LuckPerms
  • OnlyProxyJoin
  • WorldEdit (Only Creative)
  • Citizens (Only Creative)
